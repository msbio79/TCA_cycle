세포 호흡의 핵심 과정

  • 1. 피루브산 산화
    피루브산(3C) → 아세틸 CoA(2C) + CO₂
  • 2. 시트르산 합성
    아세틸 CoA + 옥살아세트산(4C) → 시트르산(6C)
  • 3. α-케토글루타르산 생성
    시트르산(6C) → α-케토글루타르산(5C) + CO₂
  • 4. 숙신산 생성
    α-케토글루타르산(5C) → 숙신산(4C) + CO₂ + ATP
  • 5. 옥살아세트산 재생
    숙신산(4C) → 옥살아세트산(4C) + FADH₂ + NADH
